Core libraries & APIs

Basis für viele Mods

Architectury API

API, die Cross-loader-Kompatibilität (Forge/Quilt/Fabric) erleichtert. Nützlich für Multi-loader Mods.
Dependencies: Fabric Loader (oder anderes, je nach Build)

Fabric API

Die zentrale API-Bibliothek für Fabric-Mods — fast immer benötigt.
Dependencies: Fabric Loader (versionsspezifisch)

Laufzeit / Helpers für Mods, die in Kotlin geschrieben sind.
Dependencies: Fabric API / Fabric Loader
